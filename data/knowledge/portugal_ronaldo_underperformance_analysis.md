# C 罗表现不佳时如何分析（学习模板）

来源：学习用分析框架，适用于“C 罗今天踢得差”“葡萄牙进攻乏力”等问题。

## 先查结构化事实

1. `get_team_profile("葡萄牙")`：看赛程、已赛场次、进球失球。
2. `get_match_analysis("Portugal")`：定位具体比赛。
3. 若结构化数据没有比分，说明比赛未结束或赛果未更新，不能编造。

## 可以讨论的维度（有证据才写）

### 1. 结果维度
- 是否输球或平局？
- C 罗是否零进球、零助攻？
- 球队整体进攻是否低效？

### 2. 角色维度
- 年龄与体能分配：关键战是否保留到下半场？
- 球队是否更依赖边路或中场推进，而非中锋终结？
- 对手是否采用低位防守，压缩中路空间？

### 3. 舆论维度
- 媒体容易把葡萄牙进攻问题简化为“C 罗状态差”。
- 更稳妥的表述是：“C 罗本场影响力低于预期，但还需要技术统计和比赛画面支持。”

## 不应直接编造

- 射门次数、跑动距离、触球热区
- “全场最低分”“被队友孤立”等没有来源的细节
- 更衣室矛盾、内讧等八卦

## 推荐回答结构

1. 比赛结果与阶段（赛前/赛后）
2. 结构化数据能确认的事实
3. 合理推测 + 明确不确定性
4. 后续赛程压力

## 与 C 罗相关的常见中文问法

- C罗今天表现怎么样
- 葡萄牙进攻为什么这么差
- C罗是不是老了
- 葡萄牙还能出线吗

这些问法应优先走 `get_team_profile` + 本文档 + 可选 `search_web`。
